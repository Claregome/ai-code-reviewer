# AI Code Reviewer – VS Code Extension

An AI-powered code review assistant built as a Visual Studio Code extension.  
This extension analyzes the currently open file and provides structured feedback on:

- Bugs
- Security issues
- Code smells
- Suggested improvements
- Overall code quality score

The project uses a locally running Large Language Model via Ollama, making it completely free and offline-capable.

---

## 🚀 Features

- One-click AI review of the active file
- Structured professional code review format
- Runs locally (no cloud dependency required)
- Privacy-friendly (code never leaves your machine)
- Lightweight VS Code extension built with TypeScript

---

## 🛠 Tech Stack

- TypeScript
- Node.js
- Visual Studio Code Extension API
- Ollama (local LLM runtime)
- LLaMA 3 model
